Multinexo para Desarrolladores

Esta sección está destinada a los desarrolladores que deseen interactactuar con la API de Multinexo.

Multinexo utiliza, principalmente, la especificación JSON API con algunas extensiones de la misma.

Autenticación

La seguridad en los accesos a Multinexo está basada en JWT.

Recursos

La API de Multinexo está pensada como una colección de recursos que pueden ser creados, editados o eliminados, cada uno accesible con un formato uniforme y basado en la especificación de JSON API.

Código abierto

Pensado teniendo en cuenta la comunidad de desarrolladores. La mayoría de las especificaciones y herramientas utilizadas son OpenSource.
